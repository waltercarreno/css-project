Gym

With this project I would like to create a website that show my Idea about a gym.

As gym user I would want a gym that allow me to adapt to my schedule.
With this website I will try to make possible this. 

As a web developer I would like to create  appealing page for prospects.

features

The website will consist of 3 pages.
The first page will be a home page that will have a hero image with a navbar.
That first page will have some text and photo that will show general features about the gym.
It will have the same footer in the 3 pages that will have contact details.


the second page will have training, seasons, sauna and other training.

 the last page will have registration form.
